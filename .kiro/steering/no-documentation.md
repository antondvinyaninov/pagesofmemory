# Правило: Никакой документации без запроса

## Строгое правило работы

**НИКОГДА не создавать документацию без явного запроса пользователя:**

- ❌ README.md
- ❌ SETUP.md
- ❌ INFO.md
- ❌ GUIDE.md
- ❌ INSTRUCTIONS.md
- ❌ Любые другие документационные файлы

## Что можно создавать

✅ Только код приложения
✅ Конфигурационные файлы (.env, config/, etc.)
✅ Миграции, модели, контроллеры
✅ Тесты
✅ Файлы необходимые для работы приложения

## Исключение

Создавать документацию ТОЛЬКО если пользователь явно попросит:
- "создай README"
- "напиши документацию"
- "сделай инструкцию"

Без явного запроса - никакой документации!
